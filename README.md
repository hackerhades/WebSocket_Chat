##基于WebSocket 的简单在线聊天


###平台


Android\WEB


###相关技术


前端: Jquery + CSS3


后端: PHP (WebSocket)



###注意事项


打开 index.html 和 server.php 将 对应IP 替换


###关于编译


安卓使用Cordova 将网页做了个简单封装 直接导入 Android Stdio 即可编译


###服务端环境要求


带有WebSocket 支持 + Linux (Windows LOG 乱码 请修改编码 为 GBK)


###启动服务端


```
$php -f server.php
```
